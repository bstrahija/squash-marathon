# 🏆 Squash Tournament Manager (24h Event)

A Laravel-based web application for managing a fast-paced 24-hour squash tournament. Built with modern tools like Livewire, Filament, and Tailwind CSS, this system streamlines player registration, match scheduling, scoring, and tournament tracking in real time.

---

## 🚀 Features

- 🧑‍🤝‍🧑 Player registration and management
- 📅 Automated match scheduling for a 24-hour tournament format
- 🏆 Live scoring and match tracking
- 📊 Tournament brackets and standings
- ⚡ Real-time UI updates using Livewire
- 🧩 Admin panel powered by Filament
- 🎨 Clean and responsive UI with Tailwind CSS
- 📱 Mobile-friendly interface for on-site usage

---

## 🛠️ Tech Stack

- 🧱 [Laravel](https://laravel.com/) – Backend framework
- ⚡ [Livewire](https://livewire.laravel.com/) – Reactive components without writing JavaScript
- 🧰 [Filament](https://filamentphp.com/) – Admin panel & dashboard toolkit
- 🎨 [Tailwind CSS](https://tailwindcss.com/) – Utility-first CSS framework

---

## 📦 Requirements

- PHP >= 8.4
- Composer
- Node.js & NPM
- MySQL / PostgreSQL / SQLite
- Laravel-compatible environment
